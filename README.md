# travel-calculator
<h1>Introduction:</h1>
A very simple tool to calculate travel budget based on well known criteria.

The current criteria items are:

1) Tickets (including taxes).
2) Accommodation.
3) Food.
4) Entertainment.
5) Transportation.
6) Gifts.
